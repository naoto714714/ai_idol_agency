## Idea 1: "Prism Narrative" (プリズム・ナラティブ)

### Design Movement

**Neo-Editorial / Soft Futurism**
雑誌のエディトリアルデザインと、近未来的なソフトな質感を融合させたスタイル。

### Core Principles

1.  **Readable Elegance**: テキストの可読性を最優先し、明朝体とゴシック体を使い分けたメリハリのあるタイポグラフィ。
2.  **Soft Light**: プリズムを通したような柔らかな光の表現をアクセントに使用。
3.  **Layered Depth**: コンテンツが重なり合うようなレイヤー構造で、物語の奥行きを表現。

### Color Philosophy

ホワイト（#FFFFFF）をベースに、光の三原色を意識したパステルピンク（#FF77B7）とシアン（#4FD8FF）を、グラデーションではなく「光の漏れ」のように使用。AIという無機質な存在に、物語という「光」が差し込むイメージ。

### Layout Paradigm

**Asymmetric Editorial Grid**
グリッドに縛られない、雑誌のような自由な配置。画像とテキストが互いに浸食し合うようなレイアウトで、視線を自然に誘導する。

### Signature Elements

1.  **Prism Borders**: 画像やカードの縁に、うっすらと虹色の光彩（border-image）を配置。
2.  **Floating Glass**: すりガラスのような質感のカード背景で、背景のストーリーを感じさせる。
3.  **Vertical Rhythm**: 縦書きの日本語テキストをアクセントとして使用し、小説らしさを強調。

### Interaction Philosophy

スクロールに合わせて要素がふわっと浮き上がるような、重力を感じさせない浮遊感のあるインタラクション。

### Animation

- ページ遷移は「光が差し込む」ようなフェードイン/アウト。
- テキストは1行ずつ、またはブロックごとに優雅に表示。

### Typography System

- 見出し: 丸みのあるモダンなゴシック体（Zen Maru Gothicなど）
- 本文: 可読性の高い明朝体（Shippori Minchoなど）で、小説のような没入感を提供。

## Idea 2: "Virtual Interface OS" (バーチャル・インターフェースOS)

### Design Movement

**Clean Cyberpunk / UI Design**
架空のOSや管理画面を模したデザインだが、ダークではなくホワイトベースで清潔感を重視。

### Core Principles

1.  **Systematic Beauty**: 情報が整理整頓された美しさ。
2.  **Micro-Interactions**: ボタンやトグルなどの細かい動きにこだわり、操作する楽しさを提供。
3.  **Data Visualization**: ストーリーの進捗やSNSのログを、データグラフのように可視化。

### Color Philosophy

ホワイト（#F7F8FB）をベースに、システムカラーとしてのピンクとシアンを使用。エラーや警告ではなく、正常稼働を示す「安心感のあるシステム色」。

### Layout Paradigm

**Dashboard Modular**
カード型のモジュールを組み合わせたダッシュボード風レイアウト。サイドバーやヘッダーが常に固定され、アプリのような操作感。

### Signature Elements

1.  **Status Indicators**: 常に点滅する小さなステータスランプや、読み込みバーのような装飾。
2.  **Monospace Accents**: 日付やIDなどに等幅フォントを使用し、デジタル感を演出。
3.  **Rounded Panels**: 角丸の大きなパネルでコンテンツを囲み、デバイス画面のような枠組み。

### Interaction Philosophy

クリック時の「カチッ」としたフィードバック（視覚的）や、タブの切り替えなど、物理的なスイッチを意識したインタラクション。

### Animation

- スライドイン、ポップアップなど、キビキビとした動き。
- ローディング時はシステム起動のような演出。

### Typography System

- 全体的にモダンなゴシック体（Noto Sans JP）。
- 数字や英字には等幅フォント（Roboto Monoなど）を混ぜてテクニカルな印象に。

## Idea 3: "Pop Diary Scrapbook" (ポップ・ダイアリー・スクラップブック)

### Design Movement

**Digital Scrapbooking / Y2K Kawaii**
手帳やスクラップブックをデジタルで再現したような、親しみやすく賑やかなスタイル。

### Core Principles

1.  **Playful Chaos**: 少し傾けたり、重なったりする「整いすぎない」楽しさ。
2.  **Tactile Texture**: 紙の質感やテープ、ステッカーのようなあしらい。
3.  **Emotional Connection**: 手書き風の要素を取り入れ、AIアイドルの「人間味」を表現。

### Color Philosophy

ホワイトノートのような背景に、マーカーで引いたような鮮やかなピンクとシアン。パステルカラーを多用し、明るく元気な印象。

### Layout Paradigm

**Freeform Collage**
グリッドを無視したコラージュ風レイアウト。写真はポラロイド風に配置し、テキストはメモ書きのように添える。

### Signature Elements

1.  **Sticker Decorations**: 星やハート、音符などのステッカー風装飾を散りばめる。
2.  **Marker Highlights**: 重要なテキストに、手書き風のマーカーラインを引く。
3.  **Washi Tape**: 画像の四隅をマスキングテープで留めたような表現。

### Interaction Philosophy

ホバーするとシールが剥がれかけたり、スタンプが押されたりするような、遊び心のあるインタラクション。

### Animation

- 要素が弾むようなバウンスアニメーション。
- スクロールに合わせて要素が回転したり、パララックスで動いたりする。

### Typography System

- 見出し: 手書き風フォントや、デザイン性の高いディスプレイフォント（Kiwi Maruなど）。
- 本文: 親しみやすい丸ゴシック体。
